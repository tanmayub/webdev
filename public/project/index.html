Some Features


How to use? Simple!
  1. Log in
  2. Add a new connection or select an already added connection
  3. Start editing!

This is as easy as it gets - no hassle of a lengthy and complex setup process. Just fire up your favorite browser and navigate to the website!


Granular Sharing

The administrator has complete control over sharing connections with non-admin users. He can grant and/or revoke access of a user for any connection.


Responsive Design

We make use of latest, industry-standard technologies and design philosophy to make sure that you can run this application on your choice of device - be it a desktop, laptop, tablet or a cell phone. The application is designed using a mobile-first approach, to ensure smooth rendering on small screen devices.
